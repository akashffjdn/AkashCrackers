# Crackers E-Commerce Platform

A full-stack e-commerce web application for browsing and purchasing crackers online, built with React, TypeScript, and modern web technologies.

## Features

- Product catalog with dynamic search and filtering
- Shopping cart with real-time price calculation
- Responsive design optimized for mobile and desktop
- Smooth animations with Framer Motion
- PDF invoice generation
- Sales analytics dashboard with Recharts
- State management with Zustand

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React 19, TypeScript, Vite |
| Styling | Tailwind CSS, Framer Motion |
| State | Zustand |
| Charts | Recharts |
| Routing | React Router 7 |
| HTTP | Axios |
| PDF | React-PDF Renderer |
| Icons | Lucide React |

## Getting Started

### Prerequisites

- Node.js >= 18
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/akashffjdnh/AkashCrackers.git
cd AkashCrackers

# Install dependencies
npm install

# Start development server
npm run dev
```

### Scripts

| Command | Description |
|---------|------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |
